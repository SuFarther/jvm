### 什么是JVM?
Java Virtual Machine - java程序的运行环境（java二进制字节码的运行环境）

### JVM好处？
1. 一次编写，到处运行的基石【重点】
2. 自动内存管理，垃圾回收功能【重点】
3. 数据下标越界检查
4. 多态，面向对象编程
